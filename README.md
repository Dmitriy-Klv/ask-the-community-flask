# Ask the Community

**Ask the Community** This is an interactive platform where users
can ask questions on any topic and receive feedback from other participants.


## 🛠 Tech Stack

- **Flask** — lightweight web framework for routing and request handling  
- **Flask-SQLAlchemy** — ORM for database interaction  
- **Alembic** — database migration management  
- **Pydantic** — data validation and serialization
